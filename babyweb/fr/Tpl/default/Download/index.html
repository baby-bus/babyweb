Q1Pourquoi le son ne fonctionne t-il pas lorsque je lance l'application ?

Ouvrir le menu "Réglages" → "Général", vérifiez l'état de l'option "Le bouton latéral sert à" → "Couper le son" puis effectuer les réglages nécessaires.

Q2Comment désinstaller une application BabyBus ?

Android：

Ouvrir le menu "Paramètres" → "Applications" ou "Gestionnaire d'applications". Appuyer sur l'application à désinstaller, puis appuyer sur "Désinstaller".

Apple：

Appuyer sur l'icône de l'application indésirable pendant trois secondes puis appuyer encore sur la croix (x).Confirmer la suppression.

Q3Comment installer les application BabyBus sur ma carte mémoire externe ?

Les applications BabyBus sont installées par défaut dans la mémoire interne de l'appareil. Pour les transférer sur la carte SD, ouvrir le menu "Paramètres" → "Applications" ou "Gestionnaire d'applications", sélectionner l'application à déplacer, puis "Déplacer vers la carte SD".

Q4Pourquoi ne puis-je plus utiliser mon application BabyBus ?

Il est possible que trop d'applications soient en cours d'exécution, ou que la mémoire disponible soit insuffisante. Fermer ou/et effacer les applications non utilisées.
